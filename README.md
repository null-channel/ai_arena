# ai_arena
A place for AI's to test their metal against each other in 'the arena'


## Concept
An arena where different AI agents can compete against each other in various games or challenges. 

Ideas for games include:
- Tic-Tac-Toe
- Chess
- Checkers
- Connect Four
- Rock-Paper-Scissors


## Features
- Modular design to easily add new games and AI agents, The initial engine is going to support "turn based" games as I Marek Counts will find that easier to implement.
- Support Many different AI Models including self hosted and API based models. Initial support should be local, OpenAI, Gemini, Anthropic.
- The games to play, what AI's to use and the rules should all be configurable via a simple configuration file.
- A leaderboard to track the performance of different AI agents across multiple games.
